Academic Progress Management for Mobile

A comprehensive mobile application for managing your academic terms, courses, and assessments.

📥 Download Latest Release

Features

🔐 Secure Authentication

User registration and login with BCrypt password hashing for maximum security.

📚 Term Management

Organize academic terms with start/end dates and manage up to 6 courses per term.

📝 Course Tracking

Track course details including instructor information, status, and notes.

✅ Assessment Management

Add up to 2 assessments per course (Objective and Performance types).

🔔 Notifications

Automatic alerts for course start/end dates and assessment due dates.

💾 Offline Storage

SQLite database for reliable offline data persistence.

Installation Instructions

  1. Download the APK - Click the download button above and get the latest WGUPathToSuccess.apk file
  2. Enable Unknown Sources - On your Android device, go to Settings → Security → Enable "Install from Unknown Sources"
  3. Install the App - Open the downloaded APK file and follow the installation prompts
  4. Launch & Register - Open the app and create your account to get started

System Requirements

  • Android 7.0 (API level 24) or higher
  • Minimum 50MB free storage space
  • Internet connection not required after installation

Technical Details

Framework: .NET MAUI 9.0
Language: C# 13
Database: SQLite (sqlite-net-pcl)
Security: BCrypt.Net-Next
Notifications: Plugin.LocalNotification
Testing: xUnit v3.2.2
